# AI工程四次跃迁

从 Prompt → Context → Harness → Loop 的 AI 工程演进框架。

## 四次跃迁

```
Prompt（语言）→ Context（信息）→ Harness（控制）→ Loop（管理）
    2022          2024-2025       2026初          2026中
```

## 公式

**Agent = Model + Harness** — 模型负责脑子，Harness 负责规矩、工具、校验、边界。

## 最小可用循环

触发 → 指令 → 状态文件 → 验证门
