# 认知资产库构建框架

作者 Sean 提出的个人知识库方法论，核心区分"资料"与"认知资产"。

## 三层架构

```
Inbox（暂存）→ Processed（加工）→ Review（沉淀）
```

## 七步闭环

收集 → 处理 → 讨论 → 沉淀（UPDATE/EXTEND/LINK/NEW）→ 审视 → 治理 → 输出

## 多角色LLM

- 处理层 = 批量引擎（Ingest）
- 讨论层 = 对话伙伴（带教/battle/质疑）
- 治理层 = 巡检员（Lint）
