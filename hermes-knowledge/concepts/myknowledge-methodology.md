# 知识治理方法论

源自 Wcof 的 myknowledge-skill 项目，是 [[../ai-education/system/wiki-harness.md|ai-education 知识库治理体系]]的蓝图。

## 四条底层原则
1. **优先解决「以后怎么找」**——分类从检索需求长出来
2. **停止重复推导，开始持续编译**——增量编译而非 RAG 式即用即走
3. **建立知识生命周期**——draft → current → archive → unknown
4. **规则才是真正的产品**——wiki-harness.md 是宪法
