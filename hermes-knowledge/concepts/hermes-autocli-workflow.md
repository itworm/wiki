# Hermes+AutoCLI+Obsidian 知识系统方案

本文是整套日报+知识库系统的**起源方案**，来自职场AI效率指南。

## 三步流程
```
信息抓取 → LLM Wiki 编译 → 微信日报
```

## 核心理念
不是技术选型，而是**系统感**——把信息抓取、整理、反馈三个环节打通，让知识库从被动容器变成主动汇报的系统。

## 与本系统映射
| 原文 | 本系统 |
|------|--------|
| AutoCLI | RSS 5源 + Tavily |
| Obsidian | llm-wiki 知识库 |
| LLM Wiki | wiki-harness 治理入库 |
| 微信日报 | AI 教育日报 |
