# AI备课方法论——先拆解课堂再生成教案

> 教师用AI备课的最大误区：一上来就让AI生成教案。正确的做法是把AI当"备课参谋"——**先拆解课堂，再生成教案**。

## 方法论核心

传统AI备课的错误姿势：直接让AI生成一份"格式完整"的教案。
问题：AI擅长生成标准格式，但真实课堂的核心不是格式，而是**学情**——学生原来会什么？可能卡在哪里？知识点到底难在哪？

**正确路径：** 把备课重心从"写教案"转移到"拆课堂"。

## 六步备课法

| 步骤 | 做什么 | 关键产出 |
|------|--------|----------|
| ① 学生困难预判 | 让AI分析学生可能误解什么、卡在哪里 | 困难清单 + 课堂支架方案 |
| ② 提炼核心问题 | 让AI帮提炼能统领整节课的关键问题 | 3个备选核心问题 |
| ③ 设计问题链 | 按观察→理解→分析→迁移→表达五层设计 | 由浅入深的问题链 + 教师追问 |
| ④ 课堂结构设计 | 判断讲—问—做—练—收束的比例 | 课堂结构建议 |
| ⑤ 课堂活动设计 | 先问三个问题（做什么/学什么/如何回归） | 3个活动方案 |
| ⑥ 生成教案初稿 | 将前五步成果汇总输出为教案 | 完整教案初稿 |

## 教案可用性五看

1. **有没有学生** — 写清学生可能的困难
2. **有没有核心问题** — 整节课围绕一个关键问题
3. **有没有问题链** — 问题由浅入深
4. **活动有没有回到目标** — 每个活动都服务学习目标
5. **有没有检测理解** — 学生能不能说出来、做出来、解释出来

## 完整备课流程

> 拆学情 → 找困难 → 定核心问题 → 设计问题链 → 安排课堂结构 → 设计活动与检测 → 生成教案初稿 → **教师本土化修改**

## 对区域教研的意义

该方法论对黄陂区电教站的直接价值：
1. 可转化为**教师AI培训的实操课程**，教教师从具体教学痛点出发使用AI
2. 六步法中的提示词可直接复用，降低教师使用AI的门槛
3. 与"项目化学习"理念一致——从真实课堂问题出发，而非从工具出发
