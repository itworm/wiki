# AI试卷讲评互动课件——飞象老师三步法

> AI试卷讲评课从"老师讲得累、学生听得困"变成"互动课"——学生不是在听答案，而是在参与、在思考、在练习。

## 方法概述

通过国家中小学智慧教育平台上的**飞象老师**AI工具，教师只需3步即可生成HTML互动试卷讲评课件：

**上传材料**（试卷+答案+评分标准） → **确认大纲** → **生成HTML互动课件**

整个流程5分钟完成。

## 三步法详解

### 第一步：上传材料
三样电子稿：测试试卷、参考答案、评分标准
平台入口：国家中小学智慧教育平台 → 人工智能教育 → 用AI → 飞象老师 → 互动课件

### 第二步：AI规划大纲
AI自动完成5步规划：目标拆解 → 环节设计 → 落地适配 → 信息确认 → 专家思考
专家思考环节包含：关键词提取、研读课标、试题分析、学情分析、目标与重难点、逐页内容设计

### 第三步：生成互动课件
最终产物是HTML互动课件，核心交互：学生点击选项 → 自动判断对错 → 弹出解析

**亮点：** 可继续对话修改，因班施教

## 实测效果（语文试卷案例）

| 功能 | 说明 |
|------|------|
| 整体分析 | 得分率、高频错题一览 |
| 错题讲解 | 分步骤呈现答题思路 |
| 变式练习 | 每道错题配1-2道同类题，即时反馈 |
| 方法归纳 | 归纳答题方法 |
| 互动元素 | 学生点击选项弹出解析 |

## 飞象老师核心能力

1. **智能试题分析** — 自动提取高频错题、重难点
2. **HTML互动课件生成** — 非静态PPT，可点击可反馈
3. **实时迭代** — 对话式修改，不满意直接改
4. **针对性训练** — 变式练习解决"一听就懂，再做还错"
5. **零技术门槛** — 上传材料说需求即可

## 对区域教研的意义

1. 试卷讲评是日常教学高频场景，该工具可直接提升讲评课效率
2. 基于国家中小学智慧教育平台，无需额外采购
3. "因班施教"理念与项目化学习、个性化教学方向一致
4. 可组织教师实操体验，作为AI教学工具培训案例
