# 乘法算理大师——三年级AI智能体课堂实录

## 课堂结构

```
前20分钟：AI一对一陪伴 + 教师巡视观察
    → 教师介入关键卡点（纸笔辅导）
    → 后20分钟：全班小组展示与梳理
```

## 核心启示

1. **AI解决了结构性问题**——过去一节课只有被叫到名字的孩子能获得学习机会，AI让每个孩子同时获得一对一陪伴
2. **教师角色转型**——从"面向全班的讲授者"变为"看见每一个孩子的设计者与陪伴者"
3. **AI的边界清晰**——AI能识别错误，但识别不了"这个孩子今天为什么犯这个错误"
4. **迭代意识**——发现AI输出太长→压缩到50字、一次只问一个问题，来自真实课堂的反馈

## 与雷刚主任"无力时助力"理念的呼应

> 要让学生自己探索和尝试，不要害怕他们遇到失败。教师只在无力时助力。

这篇案例中的教师正是这样做的：前20分钟，AI引导学生自主探索算理，遇到困难AI先尝试一轮轮追问——这是"让学生自己探索"；当学生真正卡住、AI无能为力时，教师才介入——这是"只在无力时助力"。AI的在场让"无力时助力"这个项目式学习的黄金法则，从理念变成了可操作的课堂结构。
